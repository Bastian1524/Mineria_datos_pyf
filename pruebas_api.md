# Pruebas de consumo de la API Air-Port-Codes

En este documento se evidencian las pruebas realizadas para el consumo
de la API externa Air-Port-Codes, utilizada para obtener códigos IATA
a partir del nombre de la ciudad.

## Ciudades probadas
- Madrid
- Quito
- Bogota
- Barcelona
- Frankfurt

## Resultados obtenidos
Madrid → MAD  
Quito → UIO  
Bogota → BOG  
Barcelona → BCN  
Frankfurt → FRA  

## Conclusión
La API permite transformar el nombre de la ciudad en el código IATA
necesario para sistemas de reserva de vuelos.
