# Minería de Datos – Avance 3

## Consumo de API Air-Port-Codes

En este repositorio se presenta el Avance 3 del proyecto de Minería de Datos,
donde se implementa el consumo de la API abierta **Air-Port-Codes** para
obtener el código IATA de un aeropuerto a partir del nombre de la ciudad.

Se incluye manejo de errores y un mecanismo de respaldo local para garantizar
el funcionamiento del sistema en caso de indisponibilidad de la API.

## Archivos del proyecto
- Mineria_cod_Avance.ipynb: Notebook con la implementación y pruebas de la API.
